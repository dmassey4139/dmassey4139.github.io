C#/.NET Framework


My favorite programming language right now is C#/.NET. C# was designed by Microsoft originally for use on Microsoft platforms (Windows operating system). This language is very powerful as it is maintained by Microsoft, the most profitable software company in the world. Microsoft provides C# programmers with the .NET base class libraries which provide a rich set of functionality. This means that programmers do not have to reinvent the wheel - Microsoft has already done a lot of the hard stuff and made it easier for programmers to use the classes and methods which they have designed.

Although C# was originally intended for Microsoft platforms, it has since been implemented on other platforms. One of the most notable of these is the Xamarin framework. The Xamarin framework lets you write native mobile applications for Android, iOS, and Windows phone all in C#. You are able to access 100% of the native API which normally would be accessed in Java and Swift. This greatly simplifies the mobile development process.

In 2014, Microsoft delivered a huge surprise when they introduced C# .NET core as open source. Microsfot usually does not open source anything so this was a different move for them. This has enabled the spread of cross-platform implementation such as the Mono runtime. The Mono runtime enables the same C# program to be run across Windows, MacOS, and Linux operating systems.
